Monitor

Para realizar la limpieza del monitor se necesitan los siguientes pasos

  1. Nunca ahora él para limpiarlo por dentro ya que puede sufrir Una descarga eléctrica

  2. Con un cepillo de cena fino retiré el polvo depositado en la superficie de la carcasa y la pantalla del monitor

  3. Para completar el paso anterior posee un tope limpio por la superficie de la carcasa de la pantalla del monitor

  4. Para lograr una limpieza excelente tendrá que aplicar espuma limpiadora como el monitor tiene ranuras para ventilación, la aplicación de la espuma la realizará de forma indirecta

  5. Una vez que ha terminado con la carcasa del monitor tiene que limpiar la pantalla para que lo pueda ocupar productos de limpieza de cristal, solo tiene que recordar que la aplicación también debe de llevar lo acabo de forma indirecta

Teclado

La limpieza del teclado se puede clasificar en dos tipos

  • Limpieza superficial

  • Limpieza profunda

La limpieza superficial se hace siguiendo los pasos que a continuación se listan

  1. Retira el polvo depositado en la superficie externa del teclado y con una brocha el depositado entre las teclas

  2. Aplique aire comprimido para completar el paso anterior sobre todos entre las teclas

  3. Respetar el paso 1

  4. Se ocupa espuma limpiadora para limpiar la superficie y teclas

  5. La aplicación deriva ser de forma directa para la superficie grande utilice un tapo y el caso de las teclas, aplicadores de algodón. La limpieza profunda se hará cuando se ha derramado algún líquido sobre el teclado (agua ,café ,refresco, etc) se recomienda en este caso acudir con un técnico especializado ya que podría dañar o lastimar las membranas del teclado si no se toman las medidas de de Cuidado necesario.

Raton

Para la limpieza del raton:la misma se realiza de forma superficial,utilizado para una framela en bebida de alcohol isopropilico.
